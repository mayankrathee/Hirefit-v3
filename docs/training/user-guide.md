## Core Features

### ðŸ“‹ Jobs

**Create a Job**
1. Click **"Jobs"** in the sidebar
2. Click **"New Job"** button
3. Fill in job details:
   - Title (e.g., "Senior Software Engineer")
   - Description (job responsibilities)
   - Requirements (comma-separated skills)
   - Location & salary range
4. Click **"Save"** (creates as draft)
5. Click **"Publish"** to make it active

**Job Statuses**
- **Draft**: Not visible, can edit freely
- **Open**: Accepting applications
- **Paused**: Temporarily stopped
- **Closed**: No longer accepting

### ðŸ“„ Resume Screening

**Upload Resumes to a Job**
1. Go to **Jobs** â†’ Select a job
2. Click **"Upload Resumes"** button
3. Drag & drop PDF or Word documents
4. AI automatically:
   - Parses resume content
   - Extracts candidate info
   - Scores against job requirements
   - Creates candidate profile

**Understanding AI Scores**

| Score | Meaning | Action |
|-------|---------|--------|
| 80-100 | Excellent match | Interview immediately |
| 60-79 | Good match | Consider for interview |
| 40-59 | Partial match | Review manually |
| 0-39 | Poor match | May not be suitable |

**Score Breakdown**
- **Skills Match**: How well skills align
- **Experience Match**: Relevance of work history
- **Education Match**: Degree/certification fit
- **Overall Fit**: Holistic assessment

## Tips for Best Results

### Resume Upload Tips

1. **Use PDF or Word format** - Better parsing accuracy
2. **One resume per file** - Don't combine multiple
3. **Clear formatting** - Avoid heavy graphics
4. **English language** - Currently optimized for English

### Job Description Tips

1. **Be specific** - List exact skills needed
2. **Use requirements field** - Comma-separated skills help AI scoring
3. **Include levels** - "5+ years experience" helps matching

### Team Collaboration Tips

1. **Use roles wisely** - Only admins can change settings
2. **Check activity feed** - Stay updated on team actions
3. **Share candidates** - All team members see the same candidates

## FAQ

**Q: How accurate is the AI scoring?**
A: AI scoring is a guide, not a final decision. Always review candidates personally.

**Q: Can I edit AI-generated candidate data?**
A: Yes! Go to the candidate profile and edit any information.

**Q: What file types are supported?**
A: PDF (.pdf), Word (.doc, .docx) up to 10MB each.

**Q: How do I upgrade my plan?**
A: Go to Settings â†’ Billing to see upgrade options.

**Q: Can I delete a candidate?**
A: Candidates can be archived, preserving audit trail.
